Projects Completed


1. BIKE RENT WEB APP (DEC 2022)

Developed a dynamic, responsive, and interactive web application using advanced Java technologies and APIs, consisting of over 700 lines of code. The platform enables users to seamlessly rent bikes online, with integrated document verification processes for Aadhar card and Driving license to ensure secure transactions.
The application incorporates features like real-time availability, booking management, and user authentication, providing a smooth and reliable user experience. Designed with scalability in mind.
Technologies used: Java, Spring, Hibernate, MySQL, REST APIs, HTML/CSS, JavaScript

2. E-LIBRARY WEB APP (MAY 2022)

Developed a robust eLibrary web application that enables users to browse, download, and manage books online. The platform allows administrators to efficiently add new books, manage inventory, and maintain a record of user activity and the number of registered users.
The project was designed with a focus on exploring and implementing advanced Hibernate features, which helped overcome the limitations of traditional JDBC for database management
Technologies used: Java, Hibernate, MySQL, REST APIs, HTML/CSS, JavaScript

3. Note Taker Application (DEC 2022)

Developed a user-friendly note-taking web application that allows users to create, update, and manage notes. Utilized Java Servlets for request handling and dynamic content rendering, with Spring Framework for dependency injection and overall application management.
Integrated Hibernate ORM to streamline database operations and ensure seamless interaction with MySQL.
Technologies used: Java, Hibernate, MySQL, REST APIs, HTML/CSS, JavaScript
